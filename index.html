Welcome to Smart Gesture-Based Puppet

Control a puppet in real-time using MPU6050 + ESP32 with synchronized storytelling.

Robot Details

  • Input Device: MPU6050 sensor detecting roll/pitch/yaw.
  • Processing Unit: ESP32 transmitter with BLE.
  • Output Device: Puppet with 7 servos controlling joints.
  • Storytelling: Audio playback synchronized with gestures.
